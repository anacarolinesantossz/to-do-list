# To Do List em C#

Aplicação de linha de comando desenvolvida em C# para gerenciamento simples de tarefas.

## Funcionalidades

* Adição de tarefas
* Listagem de tarefas cadastradas
* Remoção de tarefas por índice
* Menu interativo no console

## Tecnologias utilizadas

* C#
* .NET (Console Application)

## Objetivo

Esse projeto foi desenvolvido com o objetivo de praticar lógica de programação, manipulação de listas e estruturas de controle em C#.

## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/anacarolinesantossz/to-do-list.git
```

2. Acesse a pasta do projeto:

```bash
cd to-do-list
```

3. Execute a aplicação:

```bash
dotnet run
```

## Estrutura do projeto

O projeto consiste em um arquivo principal responsável pelo fluxo da aplicação, entrada de dados do usuário e manipulação da lista de tarefas.

## Possíveis melhorias

* Persistência de dados (salvar tarefas em arquivo)
* Implementação de interface gráfica
* Marcação de tarefas como concluídas
* Edição de tarefas existentes
